# The-master-retires-the-tool-goes-into-work

Single-page application built with React, TypeScript, and Vite.

## Tech Stack

- HTML
- CSS
- TypeScript
- React
- Vite

## Installation

1. Встановити залежності:

```bash
npm install

2.Запустити дев-сервер:
npm run dev

3.Зібрати для продакшн:
npm run build

4.Задеплоїти на GitHub Pages:
npm run deploy
```
